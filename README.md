A lightweight and customizable **drug harvesting system for FiveM servers**, with more features planned in future updates.

Originally built and tested on **QBox**, using **ox_lib**, **ox_inventory**, and **oxmysql**.

This is my first full script, so compatibility with other frameworks/cores is not guaranteed.

## Features
- Simple and lightweight harvesting system
- Easily customizable configuration
- Designed to be standalone
- Built with ox_lib and ox_inventory
- More drug system features planned

## Planned Features
- Drug processing
- Drug selling
- Configurable locations

## Requirements
- ox_lib
- ox_inventory
- oxmysql
- QBox (recommended core / tested)

## Download

Download the latest release from the **Releases** tab or clone the repository.

## Installation

1. Download or clone this repository.
2. Place the folder inside your server's `resources` directory.
3. Rename the folder and remove `-main` from the end if present.
4. Open `install/INSTALL.sql` using your preferred SQL tool (e.g. HeidiSQL).
5. Run the SQL file on your server database.
6. Edit `config.lua` to match your server's settings and preferences.
